  • Element UI (opens new window) PC端
  • Vant (opens new window) 有赞前端团队开源的移动端组件库

# React UI 组件库

  • Ant Design (opens new window) PC端-中后台产品
  • React Bootstrap (opens new window)
  • Material-UI (opens new window)

# 小程序组件库

  • WeUI (opens new window) 微信官方设计团队为微信内网页和微信小程序量身设计
  • Vant Weapp (opens new window) 有赞前端团队开源的微信小程序组件库
  • iView Weapp (opens new window) iView是TalkingData发布的一款高质量的基于Vue.js组件库，而iView weapp则是他们的小程序版本
  • ColorUI (opens new window) 鲜亮的高饱和色彩，专注视觉的小程序组件库
  • uni-app (opens new window) 一个使用 Vue.js 开发所有前端应用的框架，写一套代码，可发布到iOS、Android、Web（响应式）、以及各种小程序
  • uView 2.0 (opens new window) uni-app生态专用的UI框架

# 效果组件

  • Animate.css (opens new window) CSS动画库
  • Swiper (opens new window) 轮播组件
  • Mescroll (opens new window) 下拉刷新和上拉加载框架-基于原生JS
  • BetterScroll (opens new window) 解决移动端（已支持 PC）各种滚动场景需求的插件

# 工具类

  • Axios (opens new window) 可用于客户端和Node端的http请求库
  • Day.js (opens new window) 日期时间格式处理
  • Timeago.js (opens new window) 相对时间，如N小时前
  • Echarts (opens new window) 百度的可视化图表库
  • Validator.js (opens new window) 验证库
  • Meditor.md (opens new window) 开源在线 Markdown 编辑器
  • Lodash.js (opens new window) 一个一致性、模块化、高性能的 JavaScript 实用工具库

# Vue工具类

  • vue-draggable (opens new window) 基于Sortable.js实现的vue拖拽插件
  • vue-qr (opens new window) 文本转二维码
  • vue-cropper (opens new window) 图片裁剪插件
  • vue-lazyload (opens new window) 懒加载
  • vue-simple-upload (opens new window) 上传组件
